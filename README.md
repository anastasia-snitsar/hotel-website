Hotel website project created solely for educational purposes. This is not a website of an existing hotel. All images have been generated using artificial intelligence. 

The website allows users:
- to register and log in 
- book rooms
- send messages to hotel
- browse available room offers
- view the hotel’s photo gallery

Tools and software used in the project:
- HTML5 and CSS - website structure and styling
- JavaScript/jQuery - carousels, booking window, form validation, login status handling
- PHP - form processing, user registration, login system, user session management
- MySQL - database management system

The project demonstrates the use of HTML, CSS, JavaScript + jQuery, PHP, and MySQL.
It implements user registration and login with password hashing as well as room booking. 
Forms work dynamically using AJAX, allowing interaction without page reloads. 
The booking system is functional, checking room availability and dates.
